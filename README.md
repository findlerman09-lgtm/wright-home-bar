# Wright Home Bar

A standalone personal cocktail app built around the bottles and mixers in the Wright home bar.

## Features

- 335 cocktail recipes
- IBA classics, modern classics, tiki, craft, Michigan spirits, seasonal, dessert, low-ABV, and spirit-forward tags
- Bottle-based recipe browsing
- Tonight's Menu generator
- Shopping optimizer
- 1×, 2×, 4×, and 8× bartender batching
- Local ratings, notes, and favorites
- Responsive retro cocktail-book design

## Run locally

Open `index.html` in a browser, or serve the directory with any static web server.

## GitHub Pages

This repository is structured to publish directly from the root of the `main` branch.
